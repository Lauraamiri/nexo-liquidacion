# NEXO Liquidación

Sistema de Liquidación de Sueldos — parte de NEXO Suite.

Aplicación de página única (HTML + CSS + JS) para la gestión de liquidaciones
de sueldos, recibos y conceptos configurables, orientada a empresas argentinas.

## Deploy
Publicado en Vercel como sitio estático.

## Persistencia
Actualmente localStorage. Migración a Supabase en curso (ver Configuración > Conceptos de Liquidación).
